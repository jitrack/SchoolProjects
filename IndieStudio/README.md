# indie studio

Bomberman clone, 2nd year project

## how to play

  ### Player 1

- `z`: Go up
- `q`: Go left
- `s`: Go down
- `d`: Go right
- `e`: Place bomb

  ### Player 2

- up arrow: Go up
- left arrow: Go left
- down arrow: Go down
- right arrow: Go right
- shift: Place bomb

## building instructions

- `cmake .`
- `make re`
